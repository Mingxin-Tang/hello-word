# Learning Markdown
常用的Markdown语法
## 主要内容
1.# 一级标题、## 二级标题、### 三级标题...、##..## n级标题  

2.两格空格换行  

3.粗体 **()**、__()__  
**（举个例子）**  
__（举个例子）__  

4.斜体 *()*、_()_  
*(for example）*  
_(for example)_  

5.引用>  
>举例来说，**加粗字体**，*斜体*

6.有序列表  1.（空格）（内容）2. （空格）（内容）3. （空格）（内容）

7.无序列表  -（空格）（内容）、*（空格）（内容）、-（空格）（内容） 
- 第一项
  - 子项
    - 子子项
    - 子项前需要加两个空格
* 第一项
  * 子项
    * 子子项
      * ....
+ 第一项
  + 子项
    + 子子项
      + ....
1. 列表嵌套
   - 有序嵌无序
   - 1. 无序嵌有序
     2. 第二项
2. 列表嵌套第二项

8.代码块 、`(content)`  
>`hello word`这是符号表示代码块  
>
缩进表示代码块表示方法

    这是代码块  
    hello word  
    每行前面缩进四个空格
    
正常文本

9.分割线  ___  

___

>最少使用三个下划线
>
___

10.链接 `[链接文字](链接地址)`  
>例如  
>这是我的GitHub链接 [TTL的GitHub](https://github.com/Mingxin-Tang)  

11.图片 `![图片替代文字](图片路径)`  
> 一般图片放在同级文件夹或上级文件夹
>
